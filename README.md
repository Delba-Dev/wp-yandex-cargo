# wp-yandex-cargo

Минимальный плагин для **WooCommerce**: расчет тарифа **Яндекс Доставка (грузовая)** и создание заявки из заказа.

> Дисклеймер: плагин не является официальным продуктом Яндекса и не аффилирован с Яндекс. Товарные знаки принадлежат их владельцам.

## Требования

- WordPress 6.0+
- WooCommerce
- PHP 7.4+
- OAuth токен Яндекс.Доставки (B2B Cargo)
- (опционально) API‑ключ Яндекс.Карт

## Установка

1. Скачайте ZIP из **Releases**
2. WordPress → Плагины → Добавить новый → Загрузить плагин
3. Активируйте

## Настройка

1. WordPress → меню **Yandex Cargo**
2. Заполните OAuth токен, адрес/координаты склада, телефон отправителя
3. WooCommerce → Настройки → Доставка → Зоны доставки → добавьте метод **«Яндекс Доставка (грузовая)»**

## Логи (если включен Debug)

WooCommerce → Статус → Логи → источник `ycwc`
